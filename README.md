# CabLink-pwa
CabLink-booking app for Botswana with THoBoCoin (THB) .   Mobile-first Progressive Web App: book rides, simulate &amp; claim 1 THB.   Built with React/Vite, ethers.js, WalletConnect v2, Tailwind. Installable on phone.    Goal: Pilot in Gaborone—drivers get paid, riders get crypto bonus. Live preview:  Tech: React, Tailwind, PWA, BSC Testnet (chain 97)
